# Motorball

Browser multiplayer prototype with a Phaser 3 client and Colyseus TypeScript server.

## Run Locally

Start the Colyseus server:

```sh
cd server
npm run dev
```

Start the Vite client in another terminal:

```sh
cd client
npm run dev
```

Open the Vite URL in two browser tabs. Each tab controls one skater with `W` or up arrow for thrust and `A`/`D` or left/right arrows for steering.
